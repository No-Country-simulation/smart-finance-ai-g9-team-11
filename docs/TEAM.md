# Team

## Overview

Finance AI was developed by **Team 11** during the **Oracle Next Education (ONE) Hackathon**.

The project was organized into three main development areas:

- Backend Development
- Frontend Development
- Machine Learning Engineering

Each team member was responsible for a specific set of tasks while collaborating throughout the project lifecycle.
